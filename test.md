## How to install

+ Firstly you should install [nodejs](http://nodejs.org)
+ Then install it use `$ npm install -g markdown-impress`

------

+ use `------` to separate each slide
+ use comment to set impress attr, such as `<!-- x=0 y=0 rotate=0 -->`
